Praticando favicons

Para adicionar um favicon ao seu site, basta adicionar o seguinte código no head do seu site:

        <link rel="icon" href="favicon.ico" type="image/x-icon">

Se você quiser adicionar um favicon para dispositivos Apple, adicione o seguinte código:

        <link rel="apple-touch-icon" href="apple-touch-icon.png">

Para mais informações, acesse o site Favicon.io
